# Lesson 01: HSKK Intermediate Baseline Diagnosis

Status: ready

## Goal

오늘은 HSKK 중급 첫 진단 세션입니다. 아래 자료를 보고 직접 소리 내어 말한 뒤, 점수와 막힌 부분을 메모하세요.

## 30-Minute Flow

| Time | Activity |
| ---: | --- |
| 5 min | HSKK 중급 구조 확인: 듣고 반복 10문항, 그림 묘사 2문항, 질문 답변 2문항 |
| 10 min | Part 1 듣고 반복 진단 3문장 |
| 10 min | Part 2 그림 묘사형 진단 1문항 |
| 5 min | Part 3 질문 답변형 진단 2문항 + 자기 채점 |

## Part 1: 듣고 반복

아래 문장을 한 번 읽고, 화면을 보지 않고 바로 반복하세요. 실제 시험처럼 자연스러운 속도로 말하는 것이 목표입니다.

| Item | Sentence | Focus |
| --- | --- | --- |
| 1 | 我每天早上七点起床，然后坐地铁去公司。 | 시간, 순서, 성조 안정성 |
| 2 | 如果周末天气好，我想和朋友一起去公园散步。 | 조건문, 긴 문장 유지 |
| 3 | 这家饭馆的菜不但好吃，而且价格也不贵。 | 不但...而且..., 리듬 |

### Self Check

- 빠뜨린 단어가 있었나요?
- 성조가 흔들린 단어가 있었나요?
- 문장 끝까지 속도가 유지됐나요?

## Part 2: 그림 묘사형 진단

실제 그림 대신 장면 설명을 보고 1분 정도 말하세요.

**Scene**: 한 여자가 도서관 책상에 앉아 노트북으로 공부하고 있습니다. 옆에는 중국어 교재와 물병이 있고, 창밖에는 비가 오고 있습니다.

### Answer Structure

1. 장소: 这是在...
2. 인물/행동: 一个女生正在...
3. 세부사항: 桌子上有...
4. 추측/의견: 我觉得她可能...

## Part 3: 질문 답변형 진단

각 질문에 45-60초 정도 답하세요. 답변은 `직접 답변 -> 이유 -> 예시 -> 마무리` 순서로 말합니다.

1. 你为什么学习汉语？
2. 你觉得每天学习三十分钟外语有用吗？为什么？
