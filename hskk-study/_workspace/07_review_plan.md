# HSKK Review Plan

## Spaced Repetition Rule

Schedule each new speaking item for review after:

- D+1
- D+3
- D+7
- D+14
- D+30 when useful

## Review Methods

| Item type | Method |
| --- | --- |
| Sentence repetition | listen/read once, repeat without looking, compare transcript |
| Tone weakness | minimal-pair drill, then sentence drill |
| Picture description | speak using scene -> action -> detail -> opinion |
| Question answer | speak using answer -> reason -> example -> closing |
| Fluency issue | timed 45-60 second answer with fewer restarts |
