# Six-Week HSKK Intermediate Learning Plan

## Overview

- Duration: 6 weeks
- Frequency: 5 sessions per week
- Session length: 30 minutes
- Main loop: review -> new pattern -> HSKK task practice -> self-score

## Weekly Plan

| Week | Focus | Session targets | Checkpoint |
| --- | --- | --- | --- |
| 1 | Format and diagnosis | understand test, baseline speaking check, repeat basics | diagnosis report completed |
| 2 | Listen and repeat | short sentence retention, tones, rhythm | 10-sentence mini test |
| 3 | Repeat fluency and core patterns | longer sentences, repair weak tones, HSK 3-4 patterns | mock exam 01 |
| 4 | Picture description | scene order, action verbs, detail expansion | mock exam 02 |
| 5 | Question answering | opinion-reason-example structure, personal topics | mock exam 03 |
| 6 | Timed simulation | full mocks, error repair, final review | final mock and progress report |

## Daily 30-Minute Template

| Time | Activity | Output |
| ---: | --- | --- |
| 5 min | Due review | update `_workspace/07_review_plan.md` |
| 10 min | New speaking pattern or pronunciation focus | add lesson notes |
| 10 min | HSKK task practice | record answer notes/transcript |
| 5 min | Self-score and next review | update lesson and review plan |
